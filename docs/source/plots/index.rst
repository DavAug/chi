*****
Plots
*****

.. currentmodule:: chi.plots

The plots module contains a number of figure classes that allow
you to easily visualise pharmacokinetic and pharmacodynamic (PKPD) data,
as well as PKPD modelling results.

Below you can find the figure class grouped by their intended use.
At the bottom of the page, figure base classes and their API are detailed.
